## ⚠️ Ngrok Setup Fix

If you see:

```
Ngrok authtoken not found in Colab secrets.
```

Do this:

* Click 🔑 (Secrets) in left panel
* Add:

  * Name: `NGROK_AUTH_TOKEN`
  * Value: your Ngrok token
